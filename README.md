# Favoritos Linio — iOS Collection Viewer

Native iOS Swift application for browsing and managing favorite collections from Linio/e-commerce products. Built with UIKit and Swift.

## Features
- Browse product collections in a grid/list view
- View detailed collection items
- iOS-native UI with Storyboard/UIKit

## Tech Stack
- **Language**: Swift
- **Framework**: UIKit
- **Target**: iOS
- **Architecture**: MVC

## Build
Open `Favoritos.xcodeproj` in Xcode 14+ and run on simulator or device.
